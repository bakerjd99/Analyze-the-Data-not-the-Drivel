\end{floatingfigure} The \href{http://www.jsoftware.com/jwiki/Addons}{J addon} JOD now runs
on Macs. You can update
\href{http://www.jsoftware.com/jwiki/Addons/general/jod}{JOD}, or
install it for the first time, with
\href{http://www.jsoftware.com/jwiki/JAL}{JAL}. JOD now runs on all the
major J hosts: Windows, Linux and the Mac.

To keep track of host specific features I have started a series of
version documents. The Mac document is
\href{https://docs.google.com/document/d/1J-PUUB-a9UTY9gC-H26Fe8sEfQryGt5cOWRJVYBtVAw/edit?pli=1}{here}.

At the upcoming
\href{http://www.jsoftware.com/jwiki/Community/Conference2012}{J
conference}, (July 23-24, 2012), in Toronto I will show an iPhone
version of JOD. And, as a reward for attending the conference and my
little presentation, \emph{Writing Portable J addons}, I will be
raffling off hardcopy
\href{http://www.lulu.com/shop/john-baker/jod-j-object-dictionary/paperback/product-20076245.html}{printed
versions} of the JOD manual.

An updated PDF version of the JOD manual is available
\href{https://www.box.com/shared/gajfu50gc0}{here}. For more see
\href{http://bakerjd99.wordpress.com/the-jod-page/}{The JOD Page}.
